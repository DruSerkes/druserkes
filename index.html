About Me

My name is Dru, and I'm a software engineer.

While I specialize in crafting responsive UIs using React with Typescript, I also have experience building RESTful APIs (using both Node.js and Python) and working with both SQL and Document databases.

I currently work as Software Engineer for ZEPHYRx, a growing SaaS startup where I:

  • Write and advocate for clean, testable code
  • Constantly iterate on a large scale production codebase
  • Adhere to DRY principles
  • Participate in code reviews and pair programming with a fully distributed team

Prior to becoming a software developer, I spent 10 years as a professional Broadway actor, where I learned that there is always more than one way to do anything. Though I don't spend much time singing and dancing with my peers these days, my work still consists of continuously iterating on my performance to improve the quality of my product.

I am a proud graduate of Springboards' Software Engineering Career Track.
